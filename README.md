# 🐍 Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript**.
Control the snake, eat food, grow longer, and try to beat your highest score without crashing into yourself.

## 🎮 Play the Game

👉 [Play Snake Game](https://rounak-eng1.github.io/Snake-Game-/?utm_source=chatgpt.com)

## 📌 Features

* Smooth snake movement
* Score tracking system
* Simple and responsive UI
* Classic retro gameplay
* Keyboard controls
* Game over detection

## 🕹️ Controls

| Key            | Action     |
| -------------- | ---------- |
| ⬆️ Arrow Up    | Move Up    |
| ⬇️ Arrow Down  | Move Down  |
| ⬅️ Arrow Left  | Move Left  |
| ➡️ Arrow Right | Move Right |

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/rounak-eng1/Snake-Game-.git
```

2. Open the project folder

```bash
cd Snake-Game-
```

3. Run `index.html` in your browser
